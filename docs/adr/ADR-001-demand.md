# ADR-001 — kr-biz-verify 수요 검증 및 명칭

- **상태:** Accepted (수요 검증 + 명칭 확정) / 해자 갱신 방침: **차별화 강화하며 진행**
- **일자:** 2026-07-08
- **루프:** Loop 1 (수요 검증 + 명칭 확정)
- **AC:** 5개 시나리오 중 3개 이상 실수요 증거 → **4/5 달성 (통과)**

---

## 1. 맥락

Loop 1은 "누가 이걸 매주 쓰는가"를 구체화하고, 시나리오별 실수요 증거(실제 시도 사례 **또는** 유료 대체재 존재)를 확보하는 것이 목적. 본 ADR은 서브에이전트 리서치(2026-07-08, GitHub API + 공개 가격 페이지) 결과를 정리.

## 2. 시나리오 5종 × 실수요 판정

| # | 시나리오 | 강도 | 판정 | 핵심 증거 |
|---|---|---|---|---|
| 1 | 경비처리 에이전트 | **강** | 실수요 | CRNStatusChecker(엑셀 일괄), 서무실록(사무실장 실제 자동화), PublicDataReader(주간 ~2,798 DL), 간이과세자 세금계산서 의무화(2023) |
| 2 | 거래처 온보딩 | 중~강 | 실수요 | 머니핀 "B2B 가입 단순화" 명시, 유료 API 5개사 존재(건당 0~150원) |
| 3 | 이커머스 셀러 검증 | **강** | 실수요 | **전자상거래법 제21조(법적 의무)**, 공정위 통신판매사업자조회 시스템, Popbill SDK 생태계 |
| 4 | 세무 자동화 | 중간 | 실수요(간접) | NomaDamas/k-skill **6,100★**(세금계산서+진위확인 스킬), 세무사 팝빌 자동화 사례 |
| 5 | 조달/B2B 실사 | 약 | **불확실** | 나라장터 시스템 존재 only — AI 에이전트/MCP 적용 증거 미발견 |

## 3. 🚨 전략적 발견 — 계획서 해자(§0.2/§0.3) 갱신 필요

계획서는 "MCP 제품 부재"를 해자로 전제했으나, **이미 MCP 서버 5개 존재** (GitHub API 정확 데이터):

| 경쟁 MCP | 규모 | 기능 |
|---|---|---|
| `Koomook/data-go-mcp-servers` | **288★ / 50 fork** | 진위확인+상태조회+100건배치 (§1.1과 거의 동일), PyPI 등록 |
| `hjsh200219/korea-public-data-mcp` | 5★ | 34개 도구 통합, Remote MCP 배포, 가장 활발(2026-07-05 push) |
| `bamin0422/mcp_business_check_server` 외 2개 | 0~1★ | 단일 목적 래퍼 |

**함의:** "MCP 부재" 해자 붕괴. 단, 5개 경쟁 모두 **순수 래핑**이라 판정/컴플라이언스 계층이 없음 — kr-biz-verify의 잔존 차별화는 아래로 수렴:

1. **`check_invoice_eligibility` 판정 로직 + `basis` 근거** (유일한 판정 계층)
2. `explain_kr_tax_type` 해설 + 면책 문구
3. 도메인 불변 원칙 — 개인정보 무저장 · 체크섬 사전 필터 · 진위확인 캐시금지·상태조회 24h
4. 한국어 우선 제품화 + 신뢰성(가동률·에러 처리)

> **[HUMAN] 결정 필요:** 계획서 §0.2 해자 셀·§0.3 근거 갱신 여부. 본 ADR은 권장만 제시(계획서 SSOT는 [HUMAN]이 수정).

## 4. 유료 대체재 단가 — Loop 8 과금 준거

공개 단가 벤치마크 (대부분은 영업견적 비공개):

| 서비스 | 제공항목 | 단가 | 비고 |
|---|---|---|---|
| 국세청 공공데이터(원천) | 전 항목 | **무료** | 1회 100건. 모든 유료의 원천 |
| 머니핀/bizno | 사업자정보 | 검색 10원 ~ 상세 150원 | 성공 건만 과금. 월 30토큰 무료 |
| 에이픽(Apick) | 사업자정보·상태 | **0원(0포인트)** | 100건 배치=1건 과금. 대량 유리 |
| 팝빌·바로빌·쿠콘·CODEF·KRIS-R | 휴폐업·과세유형 등 | **비공개**(영업견적) | 스크래핑 안정성·SLA 차별화 |

**과금 준거 (Loop 8):**
- 상태조회 = 원천 무료·24h 캐시 가능 → **무료/저단가(건당 10~30원 마진)**
- 진위확인 = 캐시 불가·대표자명 등 민감 입력 → **프리미엄 단가 정당화**
- MCP 차별점(zod 스키마 + `basis`) = 에이픽 무료 대응에 대한 프리미엄 근거
- 전략: "상태조회 무료/저단가 + 진위확인/basis 프리미엄" 2티어 권장

## 6. 결정

1. **수요 검증 통과** (AC 4/5) → Loop 1 진행 가능.
2. **해자 갱신 권장** — "MCP 부재" → "판정+컴플라이언스+제품화" (§0.2/§0.3, [HUMAN] 수정).

4. **과금 설계** — 상태조회 무료/저단가 + 진위활력 프리미엄 2티어 (Loop 8 상세화).
